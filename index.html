📋 Guia de Instalação
  1. Apps Script → colar ERP_v3.1_CompleteScript.gs
  2. Executar installERP_v3() → aceitar permissões
  3. Implementar → Nova implementação → Aplicação web
  4. Executar como: Eu · Acesso: Qualquer pessoa
  5. Copiar o URL → colar no campo acima → Testar Ligação ✅
  6. No Google Sheets ou na interface: perfil empresa, contabilidade, AT e o bloco Folha Config (documentos, Drive, e-mail, textos PDF). Use Atualizar após mudanças na folha.
  7. Login: admin / Admin@ERP2024!
  8. Desconectar: botão ⏏️ Desconectar no topo e botão ⏏ no rodapé lateral
  9. Alterar senha e PIN obrigatoriamente no 1º login
